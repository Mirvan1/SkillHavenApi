# SkillHavenApi

## Project Overview
SkillHavenApi helps people learn from experts in different jobs. It's like asking for advice from someone who knows a lot and read their blogs.

## Technologies Used

- **.Net 7 WebApi** 
- **Entity Framework** 
- **CQRS + MediatR** 
- **SignalR** 
- **JWT Authentication & Authorization** 
- **FluentValidation** 
- **AutoMapper** 
- **Logging** 
- **Sending Mail** 
- **Localization**
